THANK YOU NEXT

Simple Webapp for Task Management Tool

Overview

Thank you next is a todolist webapp . We introduce this web application because we see that there are many students who uses todo list, so we decided to make one!

The idea is that we provide a customizeable todo list for users, and so this webapp may help students to keep track on their stuff.

Code

We coded in PHP, HTML, CSS, and abit of JS for this todo list app. Ofcourse we also established an SQL db using mariadb, and for the webserver, we used Apache 2,

Since our host machine is Linux, we installed apache2 and mariadb with "apt-get install" built in from the linux command line.
